# Документация проекта GoalTracker

## Описание

Этот репозиторий содержит документацию для веб-приложения GoalTracker - системы отслеживания личных целей.

## Структура

- `documentation.md` - исходный файл документации в формате Markdown
- `documentation.pdf` - сгенерированный PDF файл документации

## Сборка PDF

Для пересборки PDF выполните следующие команды:

```bash
npm install
npm run build-pdf
```

## Требования

- Node.js и npm для сборки PDF
- Markdown-pdf пакет установлен

## Форматирование

Документация оформлена согласно методическим рекомендациям:
- Шрифт: Times New Roman (имитируется в PDF)
- Размер шрифта: 14
- Межстрочный интервал: 1.5
- Поля: стандартные

Примечание: Для точного соответствия требованиям методических рекомендаций может потребоваться ручная настройка стилей или использование LaTeX.
