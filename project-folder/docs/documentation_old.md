# Введение

## Актуальность темы

В современном мире управление личными целями становится все более важным для достижения успеха в различных сферах жизни. Многие люди сталкиваются с проблемой организации и отслеживания своих задач, что приводит к снижению продуктивности и мотивации. Веб-приложения для управления целями помогают пользователям структурировать свои планы, мониторить прогресс и поддерживать дисциплину.

Проект GoalTracker решает эту проблему, предоставляя простой и интуитивный интерфейс для постановки, хранения и отслеживания личных целей. Приложение использует современные веб-технологии и обеспечивает сохранение данных между сессиями.

# 1. Описание предметной области

## 1.1. Понятие личных целей

Личные цели - это конкретные задачи или достижения, которые человек ставит перед собой для улучшения различных аспектов жизни: карьера, здоровье, образование, финансы и т.д.

## 1.2. Необходимость отслеживания целей

Отслеживание прогресса помогает:

- Поддерживать мотивацию
- Корректировать планы
- Анализировать эффективность
- Достигать поставленных задач

## 1.3. Существующие решения

На рынке существуют различные приложения для управления целями:

- Todoist
- Trello
- Microsoft To Do
- Evernote

Однако многие из них имеют сложный интерфейс или требуют регистрации. GoalTracker предлагает простое решение без необходимости создания аккаунта.

# 2. Анализ требований

## 2.1. Функциональные требования

Приложение должно позволять:

- Добавлять новые цели с указанием названия, категории, дедлайна, приоритета и описания
- Просматривать список целей с фильтрацией по категориям и статусу
- Отслеживать прогресс выполнения целей
- Отмечать цели как выполненные
- Просматривать статистику по целям

## 2.2. Нефункциональные требования

- Интерфейс должен быть responsive и работать на мобильных устройствах
- Данные должны сохраняться между сессиями
- Приложение должно работать оффлайн
- Код должен быть читаемым и поддерживаемым

## 2.3. Целевая аудитория

- Молодые люди 18-35 лет
- Студенты и специалисты
- Люди, занимающиеся саморазвитием

# 3. Проектирование системы

## 3.1. Архитектура приложения

Приложение построено по принципу одностраничного приложения (SPA) с разделением на модули:

- **index.html** - главная страница со списком целей
- **form.html** - страница добавления новой цели
- **stats.html** - страница статистики
- **about.html** - страница описания проекта

## 3.2. Структура данных

Каждая цель содержит следующие поля:

- id: уникальный идентификатор
- title: название цели
- category: категория (Study, Sport, Health, Finance, Self-development)
- deadline: дедлайн
- priority: приоритет (Low, Medium, High)
- progress: прогресс в процентах
- description: описание
- completed: статус выполнения
- createdAt: дата создания

## 3.3. Технологии

- **HTML5** - разметка страниц
- **CSS3** - стилизация интерфейса
- **JavaScript (ES6+)** - логика приложения
- **localStorage** - хранение данных

## 3.4. Модули системы

### 3.4.1. Модуль хранения данных (storage.js)

Отвечает за сохранение и загрузку целей из localStorage.

### 3.4.2. Модуль отображения (main.js)

Отвечает за рендеринг списка целей и обработку пользовательских действий.

### 3.4.3. Модуль формы (form.js)

Управляет формой добавления новой цели.

### 3.4.4. Модуль статистики (stats.js)

Рассчитывает и отображает статистику по целям.

### 3.4.5. Модуль утилит (utils.js)

Содержит вспомогательные функции.

Страница статистики показывает общее количество целей, процент выполнения и распределение по категориям.

Приложение имеет современный и интуитивный интерфейс с использованием иконок Font Awesome и шрифта Poppins.

Приложение протестировано в браузерах Chrome, Firefox и Safari. Все основные функции работают корректно.

## Приложение 1. Исходный код модуля storage.js

function getGoals() {
  const goals = localStorage.getItem("goals");
  return goals ? JSON.parse(goals) : [];
}

function saveGoals(goals) {
  localStorage.setItem("goals", JSON.stringify(goals));
}

function initializeGoals() {
  if (getGoals().length === 0) {
    const initialGoals = [
      {
        id: "1",
        title: "Подготовить проект по JavaScript",
        category: "Study",
        deadline: "2026-03-25",
        priority: "High",
        progress: 60,
        description: "Сделать все страницы и подключить localStorage",
        completed: false,
        createdAt: "2026-03-17",
      },
      // ... остальные цели
    ];
    saveGoals(initialGoals);
  }
}
