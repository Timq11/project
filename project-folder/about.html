О проекте

GoalTracker

GoalTracker — это веб-приложение для постановки и отслеживания личных целей. Пользователь может создавать цели, указывать их категорию, срок выполнения, приоритет и уровень прогресса. Приложение помогает удобно контролировать выполнение задач и сохраняет все изменения в localStorage.

Цель проекта

Разработать веб-приложение для постановки, хранения и мониторинга личных целей пользователя. Приложение позволяет добавлять цели, отслеживать их выполнение, управлять статусами и сохранять данные между сессиями с помощью localStorage.

Инструкция по использованию

  1. Перейдите на страницу добавления цели.
  2. Заполните форму.
  3. Нажмите кнопку «Сохранить».
  4. На главной странице можно отмечать цели как выполненные и удалять их.
  5. На странице статистики можно посмотреть общую информацию по целям.
